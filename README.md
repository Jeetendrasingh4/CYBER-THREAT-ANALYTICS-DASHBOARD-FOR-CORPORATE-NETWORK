# CYBER-THREAT-ANALYTICS-DASHBOARD-FOR-CORPORATE-NETWORK

In this project, I worked on a cyber security dataset to understand different types of attacks in a corporate network. First, I cleaned the data using Python by fixing errors and dublicate values, removing missing values, and organizing it properly.

After that, I used Power BI to create interactive dashboard where I showed important information like total attacks, types of attacks (malware, phishing, intrusion, ransomware), and high-risk incidents. I also analyzed how quickly threats are detected and resolved.

I also show a very necessary part of analytics that is forecasting of each threats .
This dashboard helps in easily tracking cyber threats and supports better decision-making to improve network security.
